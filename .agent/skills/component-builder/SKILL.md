# Component Builder Skill

## Purpose

Generate reusable UI components aligned with SalesMate AI design system.

---

## Input

- Component name
- Component purpose
- Design reference (Figma or screenshot)

---

## Output

- React component (TypeScript)
- Uses design tokens only
- Includes props interface
- Includes basic state handling

---

## Rules

- Reuse existing components first
- Do not introduce new styling patterns
- Follow UI implementation rules
- Ensure responsive behavior

---

## Output Structure

1. Component file
2. Optional index export
3. Usage example
