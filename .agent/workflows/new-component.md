# New Component Workflow

## Step 1: Check Existing Components

Search component library for reusable elements.

---

## Step 2: Confirm Design Tokens

Ensure all styling uses Figma tokens.

---

## Step 3: Build Component

- Create TypeScript React component
- Use existing UI primitives
- Avoid custom styles

---

## Step 4: Validate

Check:

- Responsiveness
- Accessibility
- Token usage
- Reusability

---

## Step 5: Export

- Export component
- Add to shared index
- Document usage
